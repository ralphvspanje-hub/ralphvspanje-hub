### How I build

I use AI agents heavily, but more as a workflow than a shortcut. New ideas go through a Claude project I set up as a virtual CTO before I write any code. It pushes back. If something survives that, it becomes an issue in Linear, then a plan, then I build it in Cursor.

The repos themselves have `CLAUDE.md` files in every directory so agents know what's there and what not to break. They also write to `AGENT_LOG.md` when something confuses them, and fix the docs on the spot. The context just keeps getting more accurate the more I use it.
